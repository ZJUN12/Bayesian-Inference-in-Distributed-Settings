# Bayesian-Inference-in-Distributed-Settings
This is a probabilistic personalized federated framework. The framework is formulated as a hierarchical Bayesian learning scheme, in which site-specific posteriors are inferred into a global consensus posterior that is recursively propagated back as an informative prior for local refinement. This posterior–prior coupling facilitates coherent uncertainty quantification while balancing cross-site knowledge transfer and site-specific adaptation.
We further introduce a divergence-based personalization mechanism that regulates the interaction between local information and shared global knowledge. The degree of global knowledge transfer is dynamically governed by the Hellinger distance between local and global posteriors, enabling data-adaptive personalization and improving predictive robustness under heterogeneous operating conditions.
